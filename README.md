# ML Algorithms Practice 

Complete implementation of core Machine Learning algorithms using Python and scikit-learn.

## Algorithms Covered

| Topic | Algorithm | Dataset |
|-------|-----------|---------|
| Classification | Logistic Regression, KNN, Naive Bayes, Decision Tree, SVM | Titanic |
| Ensemble Learning | Random Forest, Bagging, Boosting | Iris |
| Clustering | KMeans + Elbow Method | Iris |
| Dimensionality Reduction | PCA | Iris |
| Hyperparameter Tuning | GridSearchCV | Iris |

| Project | Algorithm | Accuracy |
